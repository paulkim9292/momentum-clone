이 파일은 제가 이 프로젝트를 진행하면서 새로 배운 지식들을 정리하는 공간입니다.

1. html form
- input을 이용할 경우, 사용자가 아무것도 입력을 안했을 시, 너무 길게 입력을
했을 시 경우들을 내가 직접 생각해서 코딩해야 된다.
- 그러므로 form을 이용하여 알아서 처리해주게 한다.

2. preventDefault()
- default behavior(브라우저를 refresh)를 못하게 한다.

3. localStorage API
- 사용자의 이름을 저장하는데 사용
- localStorage.setItem(key, value)
- localStorage.getItem(key)
- localStorage.removeItem(key)

4. item을 화면 중간에 배치
- html, body의 width 또는 height를 100%로 설정하고 margin 또한 0으로 설정한다.
- 중간에 배치하고자 하는 item의 width 또는 height를 100%로 설정한다.
